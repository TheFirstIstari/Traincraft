# Traincraft - A mod for Minecraft

Traincraft is being rewritten for **Minecraft 1.20.4+** using **NeoForge**.

> [!IMPORTANT]
> This is a major rewrite. The original 1.12.2 code is available on the `legacy` branch.

## Status

| Version | Minecraft | Status |
|---------|-----------|--------|
| 4.4.1_022 | 1.12.2 | Legacy |

## Development

### Prerequisites
- Java 21 (OpenJDK)
- Gradle 8.9+

### Setup
```bash
# Clone the repository
git clone https://github.com/TheFirstIstari/Traincraft.git

# Setup development environment
./gradlew setup

# Run client
./gradlew runClient
```

### Building
```bash
./gradlew build
```

## What is Traincraft?

Traincraft adds over 80 trains, wagons, and other vehicles to Minecraft, along with a new railway system featuring curved tracks and slopes.
